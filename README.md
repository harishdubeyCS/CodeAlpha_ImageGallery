# Code Alpha Image Gallery

A sleek, responsive, and interactive Image Gallery web application built using HTML5, CSS3, and JavaScript. This project was developed as part of the CodeAlpha internship task.

## 🚀 Features

- **Category Filtering:** Smooth, real-time image filtering (All, Nature, Architecture, Food) without reloading the page.
- **Interactive Lightbox (Popup View):** Click on any image to open it in an overlay/fullscreen mode.
- **Gallery Navigation:** Next (`❯`) and Previous (`❮`) buttons inside the lightbox to browse filtered images seamlessly.
- **Responsive Layout:** Built with CSS Grid and Flexbox, making it completely responsive across mobile, tablet, and desktop screens.
- **Hover Effects:** Beautiful zoom effects and overlay titles when hovering over gallery items.
- **Click-to-Close:** Close the lightbox by clicking the close (`×`) button or anywhere outside the image.

## 🛠️ Tech Stack Used

- **Frontend:** HTML5, CSS3 (CSS Grid, Flexbox, Transitions)
- **Scripting:** JavaScript (Vanilla JS - DOM Manipulation & Event Listeners)
